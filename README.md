## ✨ Features

* 🔐 Sign In / Sign Up
* 🔑 Forgot Password
* 📢 Post a Job
* 📄 View All Jobs
* 🔍 View Job Details
* 🔖 Bookmark Jobs
* 📊 Track Bookmarked Jobs (DataTable View)
* 📊 Track Applied Jobs (DataTable View)
* 📊 Track Posted Jobs & Applications
* ✅ Accept / Reject Applications
* 🔄 Job Status Updates
* 🔒 JWT Authentication for secure requests

---

## ⚠️ Note

> The following feature works only in **Local Environment**:

* 📁 View / Download Applicant CV

Vercel free plan does not allow file write operations.
You can use services like **Firebase** or **AWS S3** for production storage.

## 🛠 Tech Stack

* Next.js
* Tailwind CSS
* Redux Toolkit
* Joi Validation
* MongoDB
* SWR (for API fetching)

---

## 🔑 Environment Variables

Create a `.env` file and add the following:

```
DB_URI=your_mongodb_url
JWT_SECRET=your_secret_key
NEXT_PUBLIC_API_BASE_URL=http://localhost:3000
```

---

## 📦 Installation

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

Build for production:

```
npm run build
```

Preview production build:

```
npm run preview
```
